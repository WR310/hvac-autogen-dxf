# HVAC AutoGen DXF

Автоматизированный конвейер для перевода спецификаций систем вентиляции и кондиционирования (ОВИК) из формата PDF в готовые схемы AutoCAD (DXF).

## ⚙️ Архитектура и возможности

* **AI Парсер (`parser.py`):** Интеллектуальное извлечение данных об оборудовании из слепых PDF-спецификаций. Алгоритм умеет математически дробить сгруппированные массивы (например, 38 зонтов) на независимые ветки (от ВЕ-1 до ВЕ-38) и жестко отсекать игнорируемые элементы (крепеж, воздуховоды).
* **DXF Сборщик (`builder.py`):** Формирование геометрической взрыв-схемы трасс с использованием библиотеки `ezdxf`.
* **Умная привязка блоков:** Автоматический подбор нужного визуального блока из базового шаблона `template.dxf` (приточная установка, вытяжной вентилятор, клапан, решетка) на основе распознанного контекста.
* **Автомаркировка:** Извлечение точных марок оборудования, мощностей и параметров огнестойкости с последующей печатью в виде текстовых выносок с автопереносом строк строго под соответствующими элементами на чертеже.
